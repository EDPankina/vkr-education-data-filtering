# Алгоритм фильтрации образовательных данных

Репозиторий содержит программный прототип, разработанный в рамках выпускной квалификационной работы.

## Тема работы

Алгоритмы фильтрации данных при оценке объективности в исследованиях качества образования.

## Назначение прототипа

Прототип предназначен для предварительного выявления объектов с признаками потенциальной необъективности в результатах массовых оценочных процедур.

Алгоритм анализирует не отдельные баллы участников, а распределение результатов внутри объекта анализа: региона, муниципалитета, образовательной организации или другой группы участников.

## Используемые данные

В работе использовались два типа данных:

- синтетический набор данных для обучения и оценки моделей;
- реальный набор данных для апробации подхода и формирования статистических профилей объектов.

Реальные исходные данные не публикуются, так как использовались только для апробации подхода. В репозитории представлены синтетические данные, программный код, графики и итоговые таблицы обработки.

## Используемые признаки

Для каждого объекта рассчитываются:

- количество участников;
- средний балл;
- медиана;
- стандартное отклонение;
- минимальный и максимальный баллы;
- доля высоких результатов;
- доля низких результатов;
- признаки концентрации около пороговых значений;
- показатели пороговых скачков;
- асимметрия;
- эксцесс;
- сравнительные признаки относительно сопоставимых объектов.

## Используемые модели

В работе сравниваются следующие модели машинного обучения:

- Logistic Regression;
- Random Forest;
- Gradient Boosting;
- Isolation Forest.

Наилучший результат показала модель Random Forest.

| Метрика | Значение |
|---|---:|
| Accuracy | 0.992 |
| Precision | 1.000 |
| Recall | 0.973 |
| F1-мера | 0.986 |
